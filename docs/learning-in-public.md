# Learning in public

## Tips

- Batch work - write 2-3 short posts in one sitting instead of starting from scratch each time
- Write immediately - take notes right after finishing an assignment or fixing a bug
- Reuse content - one project can become a short post, a code snippet, and a quick summary
- Keep it short - one idea per post is enough
- Pick one platform - commit to one place instead of spreading yourself thin

## Template

### Content:
"Today I learned [concept]. I was trying to [what you were doing] and ran into [problem]. Here's how I solved it: [solution]. Link to code: [URL]"
